# Leitor de CTe - Painel Logística

Projeto em React + Vite + Tailwind para leitura de CT-e em PDF (até 30 arquivos por vez) e exportação dos dados principais para CSV/Excel.

## Rodar localmente

```bash
npm install
npm run dev
```

## Gerar build (por exemplo para GitHub Pages)

```bash
npm run build
```

O `vite.config.js` já está com `base: "./"` para funcionar em subpasta (GitHub Pages) sem ajustes.
